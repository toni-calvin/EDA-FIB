# EDA-FIB

Mi participación en el Juego "La Purga" en la asignatura de Estructura de datos y algoritmos. 

Jugador final - AIDemerzel

Funcionamiento - bfs muy simple que hace un recorrido desde la posición de cada jugador hasta un objeto que pueda ser interesante. Guarda los pasos a seguir para cada jugador. En cada iteración se calcula de nuevo. 

Compilación: Si se clona el repositorio y se intenta compilar no funcionará ya que he organziado el proyecto por directorios. Hay que cambiar las rutas internas de los ficheros y cambiar el Makefile. 

Suerte!
